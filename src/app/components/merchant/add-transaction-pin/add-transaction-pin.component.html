Ping Pin

Pingpin API enable merchant to add desired transaction pin of their choice for the completion of various financial transactions from our platform to bank.

Add Transaction Pin

You will need to send a POST request to pingPin endpoint with the following parameters

  • Your Registered Email
  • Name
  • Newpin
