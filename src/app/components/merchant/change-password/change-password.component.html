Change Password

This API allows merchant to change their password.

Change Password

You will need to send a POST request to changePassword endpoint with the following parameters

  • newpassword
  • oldpassword
  • transaction pin

Valid Parameters

To change password, you will need to provide your newpassword, oldpassword, bankcode and transaction pin.
