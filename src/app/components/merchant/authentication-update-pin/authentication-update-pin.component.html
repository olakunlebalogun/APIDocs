Merchant Reset PIN

Merchant resetPinFinal API allows merchant to reset their transaction pin.

ResetPinFinal

You will need to send a POST request to resetPinFinal endpoint with the following parameters

  • Email
  • New Pin
  • Secret Answer
  • Tokens
