Change Transaction Pin

Mercahnt need changeTransactionPin API to change their transaction pin earlier supplied.

Changing Transaction Pin of Mercahnt

You will need to send a POST request to changeTransactionPin endpoint with the following parameters

  • accountid
  • newpin
  • secretanswer
