Withdrawal of Fund

This endpoint allows merchant to initiate fund withdrawal.

Merchant WithdrawFund

You will need to send a POST request to withdrawFund endpoint with the following parameters

  • Accountid
  • Accountname
  • Accountnumber
  • amount
  • auth
  • bankcode
  • bankname
  • mac
