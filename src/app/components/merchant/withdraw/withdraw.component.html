WithdrawFund

This API allows merchant to initiate Withdrawal of fund.

Withdraw of Fund

You will need to send a POST request to withdrawFund endpoint with the following parameters.

To initiate the fund withdrwal, merchant need to provide accountname,accountnumber, amount,auth,bankcode,bankname and mac.
