verify Bvn

This API allows merchant to verify their BVN.

Bvn Verification

You will need to send a POST request to verifyBvn endpoint with the following parameters

  • Accountid
  • accountnumber
  • bankcode
  • bankname
  • bvn

Valid Parameters
