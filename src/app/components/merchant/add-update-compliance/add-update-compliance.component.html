Add Compliance

The Compliance API enables merchants to add their compliance form as requirement to become a Merchant.

Add Compliance

You will need to send a POST request to addCompliance endpoint with the following parameters

  • BVN
  • NIN
  • Proof of address
  • Means of Identification
