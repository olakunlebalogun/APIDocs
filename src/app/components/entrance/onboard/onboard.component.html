Onboard

This is the endpoint responsible for signing up an account, it is the first the merchant does to register an account

Signing Up

To access the Sign up page click here.

To be able to sign up, the following parameters must be supplied to the request

  • Business Type: This is the parameter which defines the type of business the merchant who want to sign in does.
  • Company Name: This is the name of the merchant.
  • Contact Email: The email of the merchant.
  • Contact Mobile: The mobile number of the merchant.
  • Gender.
  • id : this is the id of the merchant, and since the merchant is just signing up, the id will be 0;
  • Name : Name of the merchant.
  • Password : Password of the merchant
  • RC Number : This is the merchant RC number i.e the company registration number.
  • Referral: This is the name of the merchant's referral.
