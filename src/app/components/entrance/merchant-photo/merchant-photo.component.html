Photo API

This endpoint allows the retrieval and manipulation of image files referenced from assets.

Upload Photo

For a merchant to upload photo, the following parameters must be supplied to the request.

  • Display: This determines the appearance of the image, the default value for display is inline.
  • Filename: This is the name of the photo/logo the merchant wants to be used as a reference image and this is unique to every merchant (No two merchant can use same photo)
