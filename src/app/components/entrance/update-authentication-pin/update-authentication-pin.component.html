Update Authentication Pin

This endpoint allows merchant to update their authentication pin

This endpoint enables registered merchant to update their authentication pin by providing the following information:

Update PIN

The information below must be provided by the merchant so as to enable authentication pin update

  • Email
  • New Pin
  • Secret Answer
  • Secret Question
  • Tokens
