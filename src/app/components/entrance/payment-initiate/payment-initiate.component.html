This is the endpoint where the merchant payments starts.

Initiating Payment

For a payment to take place, the merchant must provide certain request parameters as listed below:

  • Account Number : This is the email of the merchant here the transaction will take place
  • Amount : This is the amount of money to be paid.
  • Description : A brief description of the transaction to be made.
  • Merchant ID : The id given to the merchant when the account was being setup.
  • Transaction Pin: This is the pin that will be sent to the merchant email or contact-mobile.
