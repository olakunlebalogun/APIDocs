Transaction Proration

This is the endpoint that divides the amount to be paid back based on the number of month the user selects as to when to pay back.

This endpoint has two request parameters,

  • Amount : The amount to be borrow
  • Period : The number of month(s) the user which to return, with the maximum of three(3)
