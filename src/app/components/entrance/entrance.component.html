Entrance Endpoints Documentation

This is the set of endpoints that is responsible for the user access and authentication processes.

The list of the endpoints are listed below.

  • Authentication Update (Password)
  • Authentication Update (PIN)
  • Merchant Photo
  • Sign Up
  • Login Reset
  • Password Reset
  • Payment Processing
  • Verify Payment Transaction
  • Fetch Charges
  • Fetch Transaction Proration
  • Payment Initiation
  • Payment Installment Plan
  • Payment Link
  • Resend OTP
  • Retrieve Transaction
  • Payment Verification
  • Verify Payment OTP
  • Photo Preview
  • Secret Question
  • Sign In
  • Validating CAC

Each of these endpoints has been explained in details in each of the sections

Instruction:

      Each of the requests has a response that display when the Show Response Checkbox is clicked. The default display the request cURL for each endpoint for both successful request and a failed one. The terminal is the display of the request parameter(s) for each endpoints
